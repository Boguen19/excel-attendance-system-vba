# Excel OJT Attendance Monitoring System

An Excel-based OJT Attendance Monitoring System built with VBA macros,
developed during OJT at the DPWH 2nd District Engineering Office.

## Features
- Dashboard with real-time summary (Total Students, Present, On Field, Absent)
- Daily attendance recording with AM/PM time tracking
- Search student functionality
- On Field Dashboard for tracking field dispatch and activities
- Student Database management (ID, Name, Course, School, Required/Accumulated Hours)
- Attendance Log with auto-sorting by arrival order
- On Field Log for destination and purpose tracking
- Overtime Log for extended hours recording

## Sheets
- **Dashboard** — Main attendance recording and daily log
- **OnFieldDashboard** — Field dispatch recorder
- **StudentDatabase** — Student registry and hours tracker
- **AttendanceLog** — Full attendance history
- **OnField** — On field activity records
- **OvertimeLog** — Overtime hours log

## Built With
- Microsoft Excel
- VBA (Visual Basic for Applications)
