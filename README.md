# Excel OJT Attendance Monitoring System

An Excel-based OJT Attendance Monitoring System developed during OJT 
at the DPWH 2nd District Engineering Office.

## Versions

| Version | File | Type |
|---------|------|------|
| v3 | `OJT_AttendanceSystem_v3.xlsx` | Calendar-based, 12 monthly sheets |
| Final | `OJT_Attendance_System_Final.xlsm` | Dashboard-based, VBA macros |

## Version 3 Features
- 12 monthly sheets (January – December) that auto-adjust to any year
- Student registry with required hours, school, course, and contact info
- AM/PM time entry with automatic hour calculation
- Day status system: Holiday, Half Day, Suspended, Work From Home
- Summary sheet with progress bars and completion status per student
- DTR Output sheet for printable Daily Time Record per student per month

## Final Version Features
- Real-time dashboard (Total Students, Present, On Field, Absent)
- Daily attendance recording with AM/PM time tracking
- On Field Dashboard for field dispatch and activity tracking
- Student Database management
- Attendance Log with auto-sorting
- Overtime Log

## Built With
- Microsoft Excel
- VBA (Visual Basic for Applications)
